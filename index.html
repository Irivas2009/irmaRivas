¿Qué es GIT?

Es una plataforma de desarrollo colaborativo de software
para alojar proyectos utilizando el sistema de control de
versiones Git.
El código se almacena de forma pública, aunque también se
puede hacer de forma privada, creando una cuenta de pago.
